Class: 
"Prisons"

Attributes: 
1. Number of inmates
2. Number of guards
3. Security level
4. Number of units
5. Prison type (e.g. women's or military)
6. Additional facilities
7. Operation status (e.g. defunct or working)
8. Location
9. Managing institution
10. Legality (i.e. legal or illegal) 

Public methods:
1. Put more inmates in prison
2. Change number of guards
3. Add new units
4. Change security level
5. Close the prison
6. Change management institution

Private methods:
1. Let inmates escape
2. Accept new guards
3. Become illegal
4. Become defunct
