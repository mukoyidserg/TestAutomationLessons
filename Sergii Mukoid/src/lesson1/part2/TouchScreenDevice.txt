Class: 
"TouchScreenDevice"

Attributes: 
1. Display size
2. Operating system
3. Phoning capability (yes or no)
4. Battery capacity
5. Storage capacity
6. CPU frequency
7. Body color
8. Device weight
9. Possibility to use external SD cards
10. Display resolution

Public methods:
1. Turn on the device
2. Open the main menu
3. Open an app
4. Charge battery
5. Enable/disable data exchange (when there is connection)

Private methods:
1. Turn on/off the screen
2. Load internal apps
3. Turn off when battery is discharged
4. Indicate battery charge percentage
5. Vibrate when screen is touched
